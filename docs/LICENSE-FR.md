La Licence Publique Générale Affero GNU est une licence libre, de type copyleft,
pour les logiciels et autres types d'œuvres, conçue spécialement pour garantir
la coopération avec la communauté dans le cas de logiciels exécutés sur des
serveurs en réseau.

La plupart des licences pour logiciels et autres œuvres pratiques sont
conçues pour vous retirer la liberté de partager et de modifier les œuvres.
En revanche, nos Licences Publiques Générales visent à garantir votre liberté
de partager et de modifier toutes les versions d'un programme — pour faire en
sorte qu'il reste un logiciel libre pour tous ses utilisateurs.

Les développeurs qui utilisent nos Licences Publiques Générales protègent vos
droits par deux étapes : (1) affirmer le droit d'auteur sur le logiciel, et
(2) vous offrir cette Licence qui vous donne la permission légale de copier,
distribuer et/ou modifier le logiciel.

Quand un programme est exécuté sur l'ordinateur personnel d'un utilisateur,
l'utilisateur a un accès physique au code du programme ; et si le programme est
un logiciel libre, l'utilisateur peut exécuter, copier, modifier et redistribuer
le programme, et la liberté des autres utilisateurs est préservée. Cependant,
quand un programme est exécuté sur un serveur et que des utilisateurs interagissent
avec lui à distance via un réseau, les utilisateurs ne reçoivent pas une copie
du programme. L'Affero GPL est conçue spécifiquement pour assurer que le code
source de tels logiciels serveur soit rendu disponible aux utilisateurs du serveur.

Par conséquent, nous avons conçu cette version de la GPL pour garantir que les
utilisateurs qui interagissent à distance avec le programme via un réseau se voient
aussi offrir la possibilité d'obtenir le code source du programme.
